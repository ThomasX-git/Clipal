## v0.5.2

### Highlights (EN)

- New: `clipal status` command (human-friendly summary; does not start the server).
- Improved: `clipal service status` now prints a concise summary by default, with `--raw` for full manager output and `--json` for scripts.
- Docs: updated examples to include the new status commands.

### 亮点 (中文)

- 新增：`clipal status` 命令（友好摘要输出；不会误启动服务端）。
- 改进：`clipal service status` 默认输出更简洁；增加 `--raw`（原始管理器输出）与 `--json`（脚本友好）。
- 文档：示例已同步更新，包含新的 status 用法。
